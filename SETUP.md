# FYP Setup

**Name:** Sift
**Project:** Financial Advisor Bot (CM3020 AI, template 4.2) — also planned as a commercial SaaS post-graduation.

## Stack

- **Frontend:** React Native (iOS + Android), JavaScript
- **Backend:** Supabase (auth + Postgres + storage)
- **Product framing:** educational tool / research assistant / signal generator — **not** investment advice (regulatory reasons: FCA/SEC licensing applies to personal recommendations)

## Decided but not yet built

- On-device ML inference via TFLite (model trained offline in Python, shipped via Supabase Storage)
- Python service on Modal for training + backtesting (later)
- RevenueCat for subscriptions (later)
